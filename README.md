# Sparse-Motion-Field-Tracker
Masking ROI of the video, with sparse motion field tracker

## Dependencies
- OpenCV 4.8.0
- CVUI 2.7.0 (https://github.com/Dovyski/cvui)
  -- install "cvui.h" and put into solution folder.
